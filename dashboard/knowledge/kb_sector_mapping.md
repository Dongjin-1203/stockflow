# 섹터별 매크로 민감도 & 종목 매핑 (Sector Mapping Knowledge Base)

> 이 문서는 금리·환율·공탐지수 변화에 따른 섹터 방향성 판단 기준이다.
> 개별 종목 추천이 아닌 **방향성 참고용**으로만 사용한다.

## 3. 공포·탐욕 지수별 섹터 로테이션 패턴

| 구간 | 시장 심리 | 주도 섹터 | 회피 섹터 |
|------|-----------|-----------|-----------|
| 0–24 (극단적 공포) | 패닉 매도, 안전자산 선호 | 방어주 (필수소비재·유틸리티·금), 현금 | 성장주, 소형주, 고베타 섹터 |
| 25–44 (공포) | 관망, 선별 매수 | 배당주, 저PBR 가치주 | 고밸류 성장주 |
| 45–55 (중립) | 방향성 모색 | 개별 이슈 종목 | 테마·모멘텀 |
| 56–74 (탐욕) | 위험 선호, 추세 추종 | 성장주, 테마주, 소형주 | 방어주 (상대적 소외) |
| 75–100 (극단적 탐욕) | 과열, FOMO 매수 | 테마·모멘텀 극단 | 안전자산 |

## 4. 주요 섹터 특성 요약

| 섹터 | 금리 민감도 | 환율 민감도 | 경기 민감도 | 특이사항 |
|------|------------|------------|------------|----------|
| 반도체·IT | 높음 (역상관) | 높음 (수혜) | 높음 | 글로벌 업황 사이클 중요 |
| 자동차 | 중간 | 높음 (수혜) | 높음 | 환율·원자재 동시 영향 |
| 금융 (은행) | 높음 (정상관) | 낮음 | 중간 | NIM 스프레드가 핵심 |
| 바이오·헬스 | 높음 (역상관) | 낮음 | 낮음 | 임상 결과 이벤트 중심 |
| 조선 | 낮음 | 높음 (수혜) | 높음 | 수주 사이클 2–3년 |
| 방산 | 낮음 | 높음 (수혜) | 낮음 | 지정학 리스크 연동 |
| 항공·여행 | 중간 | 높음 (피해) | 높음 | 유가·환율 동시 영향 |
| 유틸리티 | 높음 (역상관) | 낮음 | 낮음 | 배당주 성격, 안전자산 |
| 부동산·건설 | 높음 (역상관) | 낮음 | 높음 | 금리가 가장 중요한 변수 |
| 필수소비재 | 낮음 | 중간 | 낮음 | 방어주, 공탐 극단 구간 피난처 |

## 6. 사용 원칙

1. **방향성 참고 목적** — 특정 종목 매수/매도 추천 표현 사용 금지
2. **복합 요인 우선** — 금리·환율·공탐지수가 충돌할 경우 `kb_signal_rules.md` 판단 기준 적용
3. **업황 사이클 고려** — 반도체·조선 등 사이클 섹터는 매크로 외 업황 국면 병기 권장
4. **추정 명시** — 섹터 방향성이 현재 데이터로 불명확할 경우 "추정" 표기
