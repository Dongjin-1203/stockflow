## 1. 복합 신호 조합 판단표

### 1-1. 공탐지수 × 코스피 수익률

| 공탐지수 | 코스피 수익률 | 해석 | 브리핑 방향 |
|----------|--------------|------|-------------|
| 극단적 공포 (0–24) | -2% 초과 하락 | 패닉 매도 가속 — 단기 과매도 신호 | "단기 낙폭 과대 가능성. 역발상 접근 유효하나 추가 하락 리스크 병존" |
| 극단적 공포 (0–24) | 보합~상승 | 심리 대비 주가 선행 반등 — 바닥 탐색 | "심리 회복 선행 여부 확인 필요. 수급 개선 조짐 주목" |
| 공포 (25–44) | -1% ~ -2% 하락 | 조정 지속, 관망 우세 | "불확실성 지속. 방어적 포지션 유지 권고" |
| 중립 (45–55) | 보합 | 방향성 부재 | "뚜렷한 매크로 모멘텀 없음. 개별 종목 이슈 중심 장세" |
| 탐욕 (56–74) | +1% 이상 상승 | 추세 강화, 위험 선호 | "위험 선호 심리 우세. 성장주·모멘텀 중심 강세 지속 가능성" |
| 극단적 탐욕 (75–100) | +2% 초과 상승 | 과열 신호 — 차익실현 압력 | "시장 과열 구간 진입. 단기 변동성 확대 및 차익실현 가능성 주의" |
| 극단적 탐욕 (75–100) | 하락 | 심리-주가 괴리 — 급락 전조 가능 | "탐욕 지수 고점에서 주가 하락 이탈. 추세 전환 가능성 모니터링 필요" |

### 1-2. 금리 방향 × 환율 방향

| 금리 | 환율 | 해석 | 주도 섹터 |
|------|------|------|-----------|
| 인상 | 상승 (원화 약세) | 긴축 + 달러 강세 — 외국인 이탈 압력 | 방어주, 수출주 일부 (환율 수혜) |
| 인상 | 하락 (원화 강세) | 금리 인상에도 원화 강세 — 내외부 요인 분리 | 금융주 (NIM 수혜), 항공·내수 |
| 인하 | 상승 (원화 약세) | 통화 완화 + 달러 강세 — 혼재 신호 | 수출 대형주 (IT·자동차) |
| 인하 | 하락 (원화 강세) | 완화적 통화 + 외국인 유입 — 가장 우호적 | 성장주 전반, 내수 소비재 |
| 동결 | 상승 | 글로벌 달러 강세 주도 | 수출주, 방어주 |
| 동결 | 하락 | 원화 강세 — 외국인 매수 유입 가능 | 대형 성장주 |

### 1-3. 삼중 신호 조합 (핵심 시나리오)

| 시나리오 | 공탐 | 금리 | 환율 | 시장 해석 |
|----------|------|------|------|-----------|
| **Risk-On 최적** | 탐욕↑ | 인하/동결 | 원화 강세 | 전방위 강세. 성장주·외국인 매수 동반 유입 |
| **Risk-Off 최악** | 공포↓ | 인상 | 원화 약세 | 전방위 약세. 외국인 이탈, 안전자산 선호 |
| **혼재 (환율 수혜)** | 공포 | 동결 | 원화 약세 | 심리 부진 속 수출주 방어. 방향성 불명확 |
| **낙폭과대 반등 가능** | 극단 공포 | 동결/인하 | 안정 | 역발상 매수 구간 진입 탐색. 수급 확인 필수 |
| **과열 경계** | 극단 탐욕 | 인상 | 안정 | 밸류에이션 부담 + 긴축 — 고점 주의 |

## 2. 신호 충돌 시 우선순위 규칙

신호들이 서로 상충할 때 아래 우선순위로 해석한다.

```
우선순위 1 — 한국은행 기준금리 (정책 방향이 구조적 틀을 결정)
우선순위 2 — 원달러 환율 (외국인 수급과 직결, 단기 영향력 높음)
우선순위 3 — 공포·탐욕 지수 (심리 지표, 단기 모멘텀 반영)
우선순위 4 — 코스피/코스닥 수익률 (결과 지표, 위 3가지의 복합 산물)
```

**충돌 해석 원칙**
- 우선순위 높은 지표가 하락 신호 → 상위 지표 방향성 우선 서술, 하위 신호는 "단기 노이즈 가능성" 표현
- 우선순위 동급 충돌 → "신호 혼재, 방향성 불명확" 명시 후 각 시나리오 병렬 서술
- 근거 없이 한쪽으로 단정 짓지 않는다

## 3. 특수 상황 판단 기준

### 3-1. 코스피-코스닥 동반 급락 (-2% 이상)

우선 확인 순서:
1. 글로벌 지수 동향 (미국 S&P500, 나스닥 전일 방향)
2. 원달러 환율 급변 여부
3. 국내 정치·정책 이벤트 여부

→ 브리핑에 "대외 요인 vs 국내 요인" 구분 명시

### 3-2. 공탐지수와 주가 방향 역행

- 공탐↑ + 주가↓ : "심리 선행, 주가 후행 가능성. 또는 심리 지표 후행 가능성. 수급 데이터 확인 필요"
- 공탐↓ + 주가↑ : "저가 매수세 유입 가능성. 심리 회복 여부 추이 모니터링"

### 3-3. 환율 급변 (±15원 이상)

- 상승 급변 : "외환당국 구두개입 가능성 점검. 수출주 단기 수혜, 외국인 이탈 우려 병존"
- 하락 급변 : "원화 강세 속 외국인 매수 유입 가능성. 수출주 단기 환율 역풍 주의"

## 4. 브리핑 작성 규칙 (할루시네이션 가드)

### 반드시 지켜야 할 표현 규칙

| 규칙 | 허용 표현 | 금지 표현 |
|------|-----------|-----------|
| 불확실성 명시 | "가능성", "우려", "기대", "추정" | "반드시", "확실히", "무조건" |
| 근거 없는 예측 | 데이터 기반 방향성 서술 | 데이터 없는 수치 예측 ("코스피 2,700 전망") |
| 종목 추천 | "수혜 가능 섹터로 분류됨" | "OO 종목 매수 추천" |
| 추정 명시 | "현재 데이터 기준 추정" | 확정형 서술 |

### 데이터 누락 시 처리

- API 호출 실패 또는 데이터 없음 → 해당 지표 언급 생략, 나머지 데이터로 브리핑 구성
- 부분 데이터로 결론 도달 불가 → "일부 지표 미수신으로 해석 제한" 명시

### 브리핑 금지 항목

- 특정 종목 가격 예측
- 투자 손익 보장 표현
- 미확인 뉴스·루머 기반 서술
- 정치적 판단이 개입된 표현

## 5. 신호 강도 표현 가이드

브리핑 내 신호 강도를 아래 표현으로 일관성 있게 서술한다.

| 강도 | 표현 예시 |
|------|-----------|
| 강한 신호 | "뚜렷한", "확연한", "유의미한" |
| 중간 신호 | "완만한", "소폭", "일부" |
| 약한 신호 | "미미한", "제한적인", "미세한" |
| 혼재 신호 | "혼조", "방향성 불명확", "엇갈린" |
| 불확실 | "추정", "가능성", "모니터링 필요" |
