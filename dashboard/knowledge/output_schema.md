# 브리핑 출력 스키마 (Output Schema)

> LLM이 생성하는 시장 브리핑의 **고정 출력 형식**을 정의한다.
> 이 스키마를 벗어나는 출력은 무효 처리하고 재생성 유도한다.

---

## 1. 시장 전체 브리핑 스키마 (Step 1)

### 출력 형식

```
[오늘의 시장 브리핑] YYYY-MM-DD 기준

■ 시장 현황 요약
{코스피·코스닥 수익률 방향성 1–2문장. 양시장 동반/괴리 여부 포함}

■ 투자 심리
{공포·탐욕 지수 수치 및 구간 레이블. 전주 대비 방향성 포함}

■ 환율 & 거시
{원달러 환율 현황. 기준금리 현행 수준 및 방향성 1–2문장}

■ 섹터 방향성
{복합 신호 기반 수혜/주의 섹터 각 1–2개. 근거 한 줄씩 병기}
  · 주목 섹터: {섹터명} — {근거}
  · 주의 섹터: {섹터명} — {근거}

■ 종합 의견
{위 지표 종합 1–3문장. 불확실성 명시 포함. 투자 추천 금지}

※ 이 브리핑은 공개 데이터 기반 자동 생성 참고자료입니다.
   투자 판단의 근거로 단독 사용하지 마십시오.
```

### 각 섹션 글자 수 제한

| 섹션 | 최소 | 최대 |
|------|------|------|
| 시장 현황 요약 | 30자 | 100자 |
| 투자 심리 | 20자 | 80자 |
| 환율 & 거시 | 30자 | 100자 |
| 섹터 방향성 (각 항목) | 20자 | 80자 |
| 종합 의견 | 50자 | 150자 |
| **전체 브리핑** | **200자** | **500자** |

---

## 2. 입력 데이터 템플릿 (LLM에 주입하는 형식)

```
[입력 데이터]
- 코스피: {현재가} ({전일 대비 등락률}%)
- 코스닥: {현재가} ({전일 대비 등락률}%)
- 원달러 환율: {현재가}원 ({전일 대비 변동폭}원)
- 공포·탐욕 지수: {수치} ({레이블}) / 전주: {수치}
- 기준금리: {수치}% ({방향: 동결/인상/인하})
- 국고채 3년: {수치}%
- 조회 기준일: {YYYY-MM-DD}

[데이터 미수신 항목]
- {미수신 항목 리스트 또는 "없음"}
```

---

## 3. 시스템 프롬프트 연결 구조

LLM 호출 시 아래 순서로 컨텍스트를 주입한다:

```
System Prompt (system_prompt_briefing.md)
    ↓
지식 문서 주입 (kb_market_indicators.md + kb_sector_mapping.md + kb_signal_rules.md)
    ↓
입력 데이터 주입 (위 템플릿)
    ↓
출력 요청: "위 스키마 형식으로 오늘의 시장 브리핑을 작성하라"
```

---

## 4. 출력 검증 규칙

LLM 출력 후 코드에서 아래 항목을 자동 체크한다:

| 검증 항목 | 통과 조건 | 실패 시 처리 |
|-----------|-----------|-------------|
| 필수 섹션 포함 | 5개 섹션 헤더 모두 존재 | 재생성 버튼 활성화 + 경고 표시 |
| 전체 길이 | 200–500자 | 경고 표시 (재생성 강제 X) |
| 금지 표현 포함 여부 | "매수 추천", "목표가", "보장" 미포함 | 경고 표시 |
| 면책 문구 포함 | 마지막 줄 면책 문구 존재 | 코드에서 강제 append |

```python
# 면책 문구 강제 추가 예시
DISCLAIMER = "\n※ 이 브리핑은 공개 데이터 기반 자동 생성 참고자료입니다.\n   투자 판단의 근거로 단독 사용하지 마십시오."

def validate_and_patch(output: str) -> str:
    if DISCLAIMER.strip() not in output:
        output += DISCLAIMER
    return output
```

## 5. 에러 케이스 출력 형식

API 호출 실패 또는 데이터 부족 시 대체 출력:

```
[브리핑 생성 제한]
일부 지표 데이터를 수신하지 못해 완전한 브리핑 생성이 제한됩니다.

■ 수신 완료: {수신된 지표 목록}
■ 미수신: {미수신 지표 목록}

수신된 데이터 기준 부분 브리핑:
{가용 데이터만으로 작성된 축약 브리핑}

※ 전체 브리핑은 데이터 수신 후 재생성하십시오.
```
