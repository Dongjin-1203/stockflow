# 시장 지표 해석 기준 (Market Indicators Knowledge Base)

> 이 문서는 AI 브리핑 생성 시 지표 해석의 기준으로 사용된다.
> 수치만 보고 판단하지 말고, **복합 신호 조합**을 반드시 `kb_signal_rules.md`와 함께 참조할 것.

## 1. 공포·탐욕 지수 (Fear & Greed Index)

출처: Alternative.me / 0–100 범위

| 구간 | 레이블 | 시장 해석 | 대응 시사점 |
|------|--------|-----------|-------------|
| 0–24 | 극단적 공포 (Extreme Fear) | 투자자 패닉, 과매도 국면 가능성 | 역발상 매수 기회 탐색 — 단, 추가 하락 리스크 병존 |
| 25–44 | 공포 (Fear) | 불확실성 우세, 관망 심리 강함 | 방어적 포지션 유지, 낙폭과대 종목 선별적 접근 |
| 45–55 | 중립 (Neutral) | 방향성 모색 중, 변동성 낮음 | 뚜렷한 방향 없음, 개별 종목 이슈 중심 |
| 56–74 | 탐욕 (Greed) | 위험 선호 우세, 매수세 유입 | 추세 추종 유효하나 고점 리스크 점진적 증가 |
| 75–100 | 극단적 탐욕 (Extreme Greed) | 과열 국면, 과매수 신호 | 차익실현 압력 증가, 변동성 확대 주의 |

**해석 원칙**
- 단일 수치보다 **전주 대비 방향성**(개선/악화)이 중요
- 극단 구간(0–24 / 75–100)은 평균회귀 가능성 높음 — 그러나 추세 지속 가능성도 배제 불가
- 지수 급변(주간 ±15p 이상) 시 변동성 확대 경고 포함

## 2. 코스피 / 코스닥 일간 수익률

| 수익률 | 해석 | 브리핑 표현 |
|--------|------|-------------|
| +2% 초과 | 강세 (외국인·기관 동반 매수 가능성) | "뚜렷한 상승 모멘텀" |
| +1% ~ +2% | 완만한 강세 | "소폭 강세, 매수 우위" |
| -1% ~ +1% | 보합권 | "방향성 탐색, 혼조" |
| -1% ~ -2% | 완만한 약세 | "소폭 조정, 관망 우세" |
| -2% 초과 | 약세 (외국인 순매도 또는 대외 충격 가능성) | "뚜렷한 하락 압력" |
| ±3% 초과 | 급등/급락 | "비정상적 변동성 — 이벤트 드리븐 여부 확인 필요" |

**코스피 vs 코스닥 괴리 해석**

| 패턴 | 해석 |
|------|------|
| 코스피 강 / 코스닥 약 | 대형주 중심 장세 (외국인·기관 주도) |
| 코스피 약 / 코스닥 강 | 소형·성장주 선호, 개인투자자 주도 |
| 양시장 동반 강세 | 전반적 리스크 온 (Risk-on) |
| 양시장 동반 약세 | 전반적 리스크 오프 (Risk-off), 대외 변수 우세 |

## 3. 원달러 환율

기준: 장중 종가 / 전일 대비 변동 / 주요 임계값

| 구간 | 시장 해석 | 주요 영향 |
|------|-----------|-----------|
| 1,250원 미만 | 원화 강세 국면 | 수입업체 유리, 수출업체(IT·자동차) 마진 압박 |
| 1,250–1,300원 | 안정적 범위 | 뚜렷한 방향성 없음 |
| 1,300–1,350원 | 원화 약세 주의 | 외국인 이탈 우려, 수입 물가 상승 압력 |
| 1,350–1,400원 | 원화 약세 심화 | 당국 구두개입 가능성, 환헤지 비용 증가 |
| 1,400원 초과 | 달러 강세 극단 / 위기 신호 | 외환당국 시장개입 가능성 높음, 외국인 자금이탈 가속 |

**일간 변동폭 해석**

| 변동폭 | 해석 |
|--------|------|
| ±5원 이내 | 정상 변동 |
| ±5–15원 | 주의 — 뚜렷한 방향성 |
| ±15원 초과 | 급변 — 글로벌 이벤트 또는 당국 개입 가능성 |

## 4. 기준금리 (한국은행 ECOS)

현행 기준금리 방향성에 따른 시장 해석:

| 국면 | 정의 | 주식시장 영향 | 채권시장 영향 |
|------|------|--------------|--------------|
| **인상 국면** | 전회 대비 금리 상승 | 할인율 상승 → 성장주(IT·바이오) 밸류에이션 압박 / 금융주 수혜 | 채권 가격 하락, 수익률 상승 |
| **동결 국면** | 수개월 연속 동결 | 불확실성 해소, 방향성 모색 | 단기 안정, 장기 금리는 대외 변수 추종 |
| **인하 국면** | 전회 대비 금리 하락 | 성장주 재평가, 리츠·배당주 수혜 | 채권 가격 상승, 기간물 중심 강세 |
| **인하 기대** | 시장 선반영 | 선행 매수 유입, 실제 인하 시 "Buy the rumor, Sell the fact" 가능 | 장기물 금리 선행 하락 |

**기준금리 수준별 해석 (2024–2026년 컨텍스트)**

| 수준 | 해석 |
|------|------|
| 2.0% 미만 | 초저금리 — 경기부양 국면 |
| 2.0–2.5% | 완화적 통화정책 |
| 2.5–3.0% | 중립금리 근방 |
| 3.0–3.5% | 긴축적 — 성장주 부담 구간 |
| 3.5% 초과 | 고금리 — 가계부채·부동산 압박 심화 |

## 5. 국고채 3년 수익률 (ECOS)

|------|------|
| 기준금리 대비 -0.3%p 이하 | 경기침체 우려 반영 (장단기 역전 경계) |

| 기준금리 대비 +0.5%p 이상 | 인플레이션 우려 또는 재정 리스크 프리미엄 |

|----------|------|
| 양수 확대 | 경기 회복 기대, 정상 수익률 곡선 |

| 음수 (역전) | 경기침체 선행 지표 — 역사적으로 6–18개월 후 침체 |

## 6. 해석 공통 원칙

1. **단일 지표로 결론 내리지 않는다** — 반드시 복합 신호 조합 참조 (`kb_signal_rules.md`)
2. **방향성 > 절대 수준** — 전일/전주 대비 변화가 현재 수준보다 중요
3. **근거 없는 예측 금지** — 데이터에 없는 내용은 "추정" 명시 또는 언급 생략
4. **컴플라이언스 가드** — 특정 종목 매수/매도 추천 표현 사용 금지
